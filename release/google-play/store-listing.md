## Full description

Kojori Bus is a focused transit app for traveling between Kojori and Tbilisi.

Use it to:
- check live and scheduled departures for routes 380 and 316
- switch between directions quickly
- view nearby stops and route maps
- keep using baked timetable data when network coverage is poor
- place a home-screen widget for quick glance updates

The app is designed to stay simple and fast:
- no account required
- no ads or analytics
- no personal data sent to our servers
- optional location access for map and direction features

Transit data comes from the TTC public API and may change or become unavailable at any time.
