# App access notes

The app does not require login or special credentials.

## Reviewer instructions

1. Install the production build.
2. Open the app normally.
3. Deny location permission if requested; the main app still works.
4. Optionally allow location to verify map and direction detection.

## What to test

- departures screen loads
- timetable screen loads
- map screen opens
- settings can be changed
- widget configuration is not required for review
